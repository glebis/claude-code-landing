Как устроена неделя

Каждая неделя — это две живые встречи в Zoom (вторник и суббота) плюс один коворкинг по средам. Между встречами — 3–5 часов самостоятельной работы над своим проектом, поддержка в чате Telegram, доступ ко всем материалам и записям.

Все встречи веду я лично — Глеб Калинин. 20+ лет в IT, бывший CPO Ozon.travel, последние два года — преподавание агентных подходов и коучинг продактов и предпринимателей.

Три формата

Лаборатория сознательно избегает «лекционного» формата. Большая часть знаний приходит через практику и разбор реальных кейсов участников.

Вторник 18:00 CET · 90 мин

Вечер знаний

Тема недели + демонстрации + разбор кейсов участников. Я показываю на экране, как решаю задачу, отвечаю на вопросы по ходу. Это не лекция — скорее парное программирование с группой.

Суббота 10:00 CET · 90 мин

Практика и Q&A

Live coding клиника: участники приносят свои задачи, мы решаем их в реальном времени. Это место, где «3 часа отлаживал — за 15 минут вместе разобрали».

Среда 09:00 CET · 3 × 25 мин

Коворкинг (опционально)

Три помодоро подряд с короткими перерывами. Включаешь камеру, объявляешь свою цель на 25 минут, работаешь в тишине. Между раундами — короткий обмен «что застряло». Лучшее средство против прокрастинации.

Telegram-чат · 24/7

Я в чате каждый день. Ответы на вопросы, разбор скриншотов, ссылки на свежие практики. Чат остаётся доступным после окончания когорты.

Что мы делаем на live-сессиях

Реальные задачи из прошлых когорт. Участники приходят со своими проектами, я не готовлю синтетические упражнения.

YouTube → research pipeline

Скачать субтитры через yt-dlp, извлечь темы, параллельно отправить субагентов на исследование каждой. Финальный отчёт в Markdown.

KPI-калькулятор для отдела продаж

Полноценное Next.js-приложение: синхронизация с PrintEffect API, расчёт бонусов, ролевые дашборды. Собрано за две сессии.

Telegram-бот на Claude Agent SDK

Бот, который принимает голосовые, превращает их в задачи и кладёт в Notion. Полный цикл: SDK → деплой → подключение к чату.

Конкурентный анализ через Firecrawl

Скрейпинг сайтов конкурентов, извлечение цен и фич, генерация сравнительной таблицы. Запускается по расписанию каждую неделю.

Создание собственного skill-а

Упаковка повторяющегося рабочего процесса (например, генерация отчёта по клиенту) в переиспользуемый Claude Code skill с тестами.

Презентация из заметок Obsidian

Markdown → Marp-слайды → PDF → озвучка через ElevenLabs → деплой на Vercel. Один промт — готовый артефакт.

Транскрипция и анализ встреч

Подключение Fathom + локальный анализ: автоматическое извлечение решений, action items, цитат клиента в CRM.

Personal memory system

Файловая память для агента: профиль, обратная связь, контекст проекта. Чтобы агент помнил вас от сессии к сессии.

MCP-сервер под свой инструмент

Обёртка внутреннего API/базы в MCP-сервер, чтобы любой агент мог им пользоваться. От нуля до работающего сервера за встречу.

Hooks для автоматизации

Pre/post-tool hooks: автокоммит, валидация, нотификации в Telegram. Превращают Claude Code из инструмента в настроенную среду.
